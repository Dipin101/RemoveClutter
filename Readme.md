# Remove Clutter App

A simple **Node.js utility** to automatically organize and declutter your folders by file type, making file management faster and more efficient.

---

## Features

- Scans a specified folder or directory for files.
- Automatically creates folders based on **file type or category**.
- Moves files into their respective folders for a clean and organized structure.
- Reduces clutter and saves time on manual organization.
- Simple command-line interface for easy use.

---

## Technologies Used

- **Node.js** – backend logic and file system operations
- **fs module** – file system management
- **path module** – organize files by type and location
- **GitHub** – version control
